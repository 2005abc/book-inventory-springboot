# Book Inventory System (Spring Boot + Thymeleaf + MongoDB)

**Short description**  
A simple Book Inventory System built with Spring Boot, Thymeleaf and MongoDB. Supports adding, searching and deleting books and listing available books.

## Features
- Add new books via a Thymeleaf form
- Search book by `bookid`
- Delete book by `bookid`
- View all books and availability
- Persists data in MongoDB

## Tech Stack
- Java 21
- Spring Boot
- Spring MVC + Thymeleaf
- Spring Data MongoDB
- MongoDB (Compass for GUI)
- Gradle
- Bootstrap (frontend)

## Prerequisites
- Java JDK installed 21
- Git
- MongoDB running locally 
- Gradle wrapper is included in the repo
