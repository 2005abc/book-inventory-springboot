<p>
A simple <strong>Book Inventory System</strong> built using 
<strong>Spring Boot, Thymeleaf, and MongoDB</strong>.
The application allows users to manage books by adding, searching, deleting,
and viewing available books.
</p>

<hr>

<h2>ðŸš€ Features</h2>
<ul>
    <li>Add new books using a Thymeleaf-based form</li>
    <li>Search books by <strong>Book ID</strong></li>
    <li>Delete books by <strong>Book ID</strong></li>
    <li>View all available books</li>
    <li>Data persistence using MongoDB</li>
</ul>
